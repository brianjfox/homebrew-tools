# Brian J. Fox — Homebrew Tap

Homebrew formulae for tools maintained by Brian J. Fox.

## Usage

```sh
brew tap brianjfox/tools
brew install gnash
```

(You only need `brew tap brianjfox/tools` once; afterwards `brew install <tool>`
and `brew upgrade <tool>` work as usual.)

## Formulae

- **gnash** — a modular C++20 reimplementation of GNU Bash 5.3 with multiple
  shell personalities (bash, zsh, ash, ksh, and a full csh/tcsh interpreter).
  <https://github.com/brianjfox/gnash>
